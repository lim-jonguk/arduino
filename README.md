# arduino-weather_lcd

esp32 보드를 이용하여 lcd보드를 연결하고 \newline
공공데이터인 시간을 얻기 위해 
world time API
기상청 날씨정보RSS 
한국환경공단 에어코리아 대기오염정보를 가져와
미세먼지는 RGB LED를 통해
날씨와 시간은 lCD에 표시함
